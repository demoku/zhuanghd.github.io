关于博客：

“三味书屋” 搭建于 2016 年 7 月。由于 GitHub Pages 是免费的静态站点，具有以下特点：免费托管、自带主题、支持自制页面、并且可以绑定自己的域名，非常适合平时写写文章，所以我决定使用 GitHub Pages 来搭建自己的博客。 博客基于开源的 Hexo 主题 NexT.Pisces，并在此基础上做了一些个性化定制。博客主要以分享读书心得和 IT 技术为主，部分文章出自转载，当作是记录的笔记来阅读，这些来自网络，也将回馈给网络上有所需要的朋友，希望对您有所帮助。
